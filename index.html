View Project

Sparse view 3D reconstruction with Gaussian splatting

This project aims to enhance Gaussian splatting's performance and accuracy with sparse-view inputs, reducing the need for numerous multi-view images, which can be cumbersome. Efficiently reconstructing high-quality 3D objects from a minimal number of images is valuable for accelerating applications such as 3D asset creation for games, movies, and AR/VR products. This work is crucial for these applications. This project won top 3 out of 80 projects.
